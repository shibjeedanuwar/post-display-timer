Display posts with a countdown timer and control how long visitors view each post before proceeding to the next one.

== Description ==

Post Display Timer is a powerful WordPress plugin that allows you to control how long visitors spend reading each post on your website. Perfect for research studies, timed reading exercises, or controlled content presentation.

* Mobile-friendly and responsive

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/post-view-timer` directory, or install the plugin through the WordPress plugins screen directly
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to 'Post View Timer' in your WordPress admin menu to configure the settings

== Frequently Asked Questions ==

= Can I customize the timer duration? =

Yes, you can set any custom duration in seconds for how long each post should be displayed.

= Does it work with any post type? =

Yes, the plugin works with standard posts and custom post types.

= Can I prevent users from opening multiple tabs? =

Yes, there's a built-in feature to prevent users from viewing posts in multiple tabs simultaneously.

= Is it possible to randomize the post order? =

Yes, you can enable random post ordering in the plugin settings.
